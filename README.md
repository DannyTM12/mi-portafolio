Este repositorio contiene el código fuente de mi portafolio web personal. Diseñado y desarrollado desde cero para mostrar mis proyectos en ingeniería de software, inteligencia artificial, ciberseguridad y docencia.

## ✨ Características Destacadas

* 🌓 **Modo Oscuro/Claro Nativo:** Implementado con variables CSS (`:root`) y guardado de preferencias en `localStorage`.
* 📱 **Diseño Mobile-First:** Totalmente responsivo, adaptándose desde pantallas móviles hasta monitores de escritorio con Flexbox y CSS Grid.
* 🚀 **Carga Ultrarrápida (Zero Dependencies):** Construido sin frameworks externos. Puros HTML semántico, CSS moderno y JavaScript Vanilla.
* 🎭 **Animaciones al Scroll:** Uso de `IntersectionObserver` para revelar elementos suavemente a medida que el usuario explora la página.
* 🎨 **Arquitectura CSS Escalable:** Uso de la metodología BEM (Block Element Modifier) para mantener las clases organizadas y prevenir colisiones de estilos.
